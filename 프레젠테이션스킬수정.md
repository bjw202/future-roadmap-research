# 프레젠테이션 병렬 빌드 파이프라인 에러 방지 개선

Context

2026-03-25 프레젠테이션 생성 시 병렬 Slide Builder 간 API 불일치로 4가지 런타임 에러 발생:

1. COLORS 키 축약 (accent_blue → accent)
2. addCard() 시그니처 불일치 (객체 vs 위치 인자)
3. addTitledTable() 인자 순서 오류
4. 합치기 시 sed 줄번호 기반 코드 절삭 + 후속 파트가 다른 파트의 함수명을 추측

근본 원인: 후속 파트 에이전트가 Part 1의 실제 상수/함수 정의를 보지 못한 채 추측으로 코드 작성. 프롬프트에 금지사항과 정확한 API 계약이 부족.

목표: 파트 수(2\~5+)에 관계없이 안전하게 병렬 빌드할 수 있도록 스킬/에이전트 설정 수정.
